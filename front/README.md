## Getting Started

### Prerequisites

Install **[npm](https://www.npmjs.com/get-npm)** and **[node](https://nodejs.org/en/download/package-manager/)**

### Installing

Install dependencies

```bash
$ npm install
```
### Run

#### Stable API

```bash
$ npm run stable
```

#### Beta API

```bash
$ npm run beta
```

#### Unstable API

```bash
$ npm run unstable
```

#### Local API

```bash
$ npm run local
```
